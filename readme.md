# ART-1 Neural Network from scratch

This project implements an **ART1 (Adaptive Resonance Theory)** neural network for unsupervised clustering and evaluates its performance on the **Dry Bean Dataset** using different binary feature encoding techniques.

The objective is to investigate how different data representations affect ART1 clustering quality under different vigilance parameter values.

This project was developed as part of the undergraduate **Computer Science** program at the **State University of Ceará (UECE)** for academic purposes, focusing on the study of neural networks, unsupervised learning, and clustering techniques.

## Dataset

The experiments use the **Dry Bean Dataset** from the UCI Machine Learning Repository.

### Dataset Characteristics

- Samples: 13,611
- Classes: 7 bean varieties
- Features: 16 numerical attributes
- Type: Multiclass classification dataset

The dataset is automatically downloaded using the `ucimlrepo` package.

---

## ART1 Neural Network

ART1 is an unsupervised neural network designed for binary input patterns.

Its main characteristics are:

- Incremental learning
- Dynamic category creation
- Stability-plasticity balance
- No need to predefine the number of clusters

The network creates new categories whenever an input pattern does not satisfy the vigilance criterion of existing categories.

---

## Data Preprocessing

Since ART1 requires binary inputs, three encoding strategies were evaluated:

- **Binary Thresholding:** Converts each sample into a binary vector based on its median value.
- **Quartile-Based Discretization:** Divides features into quartiles and applies one-hot encoding.
- **Thermometer Encoding:** Represents feature values using cumulative binary levels, preserving ordinal information.

All representations were processed using **Complement Coding**, a technique commonly employed in ART networks to improve category formation and stability.

---

## Experimental Setup

The ART1 network was evaluated using vigilance values (p) ranging from **0.1 to 0.9**.

- Low vigilance → fewer and more general clusters.
- High vigilance → more and more specialized clusters.

---

## Evaluation Metrics

Clustering performance was assessed using:

- Adjusted Rand Index (ARI)
- Normalized Mutual Information (NMI)
- Purity
- Homogeneity
- Completeness
- V-Measure
- Fowlkes-Mallows Index (FMI)

Additionally, the number of categories generated by ART1 was analyzed.

---

## Visualization

For each experiment, a **PCA projection** was used to visualize and compare:

- Ground-truth classes
- ART1-generated clusters

The resulting figures are stored in the `results/` directory.
